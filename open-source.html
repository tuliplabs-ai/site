Tulip

The open-source SDK for building grounded cybersecurity agents — agents that audit cloud posture, triage alerts, and fingerprint AI infrastructure, and only ship a finding when the evidence clears the bar.

Read the docs → GitHub

pip install tulip-agents

Findings you can trust, by construction

Most agent frameworks give you an answer and ask you to trust it. Tulip is built the opposite way: a security Finding can only be produced from evidence that clears the grounding threshold. Below the bar, the agent abstains and records why — so an ungrounded finding is a false positive by construction. There's no public path that builds one.

That's the difference between an AI-security SDK and an agent framework with a security demo. The same Agent class scales from a five-line script to a multi-agent SOC, and the whole time it narrates itself: every reasoning step, tool call, and hand-off is a typed event you can stream, replay, and test against.

What's inside
Grounding

Findings backed by evidence

A security finding is admitted only when its evidence clears the GSAR threshold; below the bar it abstains, with an audit trail. Tagged to MITRE ATLAS / OWASP, ready for a SIEM.

Security tools

Read-only by construction

The cloud-posture auditor literally can't write — its tools admit only describe/list/get calls, with the IAM identity as the backstop. Pair guardrails and a model-judge on every tool call.

Coordination

Eight shapes, one class

Single agent, pipelines, loops, orchestrator + specialists, swarm, hand-off, and cross-process A2A — composed from the task, not hand-wired.

Observability

Typed event stream

An opt-in event bus surfaces 40+ canonical events from every layer through one context — no external broker — with an OpenTelemetry hook when you want it.

Durability

Pause & resume

Idempotent tools so a risky action never fires twice; checkpoint to Redis, Postgres, or S3; interrupt for human approval and resume exactly where it stopped.

Reach

Provider-agnostic

OpenAI, Anthropic, self-hosted vLLM, or any OpenAI-compatible endpoint via base_url. RAG, MCP client + server, and a FastAPI agent server included.

The Workbench

A self-contained playground for every Tulip pattern — vanilla-TS UI, a thin Node proxy, and the SDK runner behind it. Bring your own provider key and watch a multi-agent run emit its event stream live. Run it on localhost in three terminals, or as a single Docker container.

# a grounded cloud-posture audit
from tulip.security import create_soc_analyst, ground_report, is_finding

analyst = create_soc_analyst(model="anthropic:claude-sonnet-4-6")
result = await analyst.run("Audit this account's IAM posture.")

for f in ground_report(result.output):
    if is_finding(f):   # only ships when evidence clears the bar
        print(f.severity, f.title, f.gsar_score)
